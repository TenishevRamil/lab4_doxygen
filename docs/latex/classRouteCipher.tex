\doxysubsubsection{Подробное описание}
Шифрование методом маршрутной перестановки 

Класс реализует шифр табличной маршрутной перестановки. Маршрут записи\+: по горизонтали слева направо, сверху вниз. Маршрут считывания\+: сверху вниз, справа налево. \begin{DoxyWarning}{Предупреждения}
Ключ (количество столбцов) должен быть положительным числом 
\end{DoxyWarning}
